3. Testy

3.1 Niezawodność sieci

Za badaną statystykę przyjmujemy szacunkową wartość niezwodności sieci.

Niezawodność zależna jest od kilku parametrów:

  • TmaxT_{max}Tmax​ - maksymalne opóźnienie pakietu,
  • 1−p1 - p1−p - prawdopodobieństwo uszkodzenia sieci,
  • m - wielkość pakietu, zmnieszająca stałą b (patrz: 2.4).

Wyliczamy ją według następującego algorytmu:

    1. Dla k powtórzeń
    • 1.1. Dla każdej z krawędzi w danej sieci
      • 1.1.2. Wylosuj jednostajnie liczbę rand z przedziału [0, 1)
      • 1.1.3. Jeśli p < rand, usuń obecną krawędź
    • 1.2. Jeśli nowo powstały graf nie jest spójny, przerwij iterację
    • 1.3. Dla każdej z krawędzi danej sieci
      • 1.3.1. Jeśli c(e)m<a(e)\frac{c(e)}{m} < a(e)mc(e)​<a(e) - krawędź jest zapchana, przerywamy iterację
    • 1.4 Policz średnie opóźnienie sieci ze wzoru:
      • T=1∣N∣⋅∑e∈Ea(e)c(e)m−a(e)T = \frac{1}{|N|}\cdot\sum_{e\in{E}}\frac{a(e)}{\frac{c(e)}{m} - a(e)}T=∣N∣1​⋅∑e∈E​mc(e)​−a(e)a(e)​, gdzie ∣N∣|N|∣N∣ to suma wartości z macierzy natężeń
    • 1.5 Jeśli T<TmaxT < T_{max}T<Tmax​ dane powtórzenie jest sukcesem
    1. Zwróć średnią arytmetyczną liczby sukcesów dla k powtórzeń.

Pseudokod:

  function networkReliability(net, T_max, p, m, k)
    successes <- 0
    repeat k times
      for edge in edges(net)
        if (p < random(U[0,1)))
          remove_edge(edge)
      if (net.graph is not connected) continue
      for edge in edges(net)
        if (c[edge] / m < a[edge])
          break
      T <- averagePacketDelay(net)
      if (T < T_max) successes <- successes + 1
    return successes / k
